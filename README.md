# 🐾 FurrSure – Pet Adoption, Daycare & Food Platform

A full-stack MERN web application designed to provide an all-in-one platform for pet care services, including pet adoption, daycare booking, and an online pet food store.

---

## 📌 Overview

FurrSure is a comprehensive pet care platform that connects users with pets through adoption, provides daycare services, and allows users to purchase pet food — all in one place. The system also includes an admin dashboard for managing users, pets, and products.

---

## ⚙️ Tech Stack

**Frontend:**
- React.js
- React Router
- Axios

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Mongoose)

**Authentication & Security:**
- JWT (JSON Web Tokens)
- bcrypt (password hashing)
- OTP Email Verification (Nodemailer)

---

## ✨ Features

### 👤 User Features
- User Registration & Login with OTP verification
- Browse pets available for adoption
- View pet details (age, breed, vaccination, etc.)
- Submit adoption requests
- Book daycare services (daily/weekly/monthly)
- View real-time daycare availability
- Browse and purchase pet food
- Add items to cart and checkout
- View adoption history

---

### 🛠️ Admin Features
- Manage pet listings (add/edit/delete)
- Approve or reject adoption requests
- Manage pet food products
- Monitor users and system activity
- Handle daycare check-in/check-out

---

### 🌐 General Features
- Dark/Light mode
- Responsive UI (mobile + desktop)
- About Us page
- Secure authentication system

---

## 🏗️ Architecture

The application follows a typical MERN architecture:

- **Frontend (React)** → handles UI and user interaction  
- **Backend (Express/Node)** → handles API and business logic  
- **Database (MongoDB)** → stores user, pet, and product data
